# 하남돼지집 서호점(Tây Hồ) 공식 홍보 랜딩페이지 — 설계 문서

- **작성일**: 2026-07-09
- **목적**: 하노이 거주 베트남인을 타겟으로 한 하남돼지집 서호점(West Lake Hanoi) 브랜드 인지도 + 신규 유입 확대
- **배포**: Vercel (URL 생성)

---

## 1. 목표 & 타겟

### 캠페인 목표
- **인지도(Brand Awareness)**: 글로벌 K-BBQ 1위 하남돼지집을 베트남 시장에 각인
- **신규 유입(New Acquisition)**: 광고 → 랜딩 → 방문(길찾기/예약/문의) 전환

### 타겟
- 하노이 거주 베트남인, 25–49세, 성별 무관
- 방문 목적: **가족 식사** & **비즈니스 접대**

### 광고 소재 두 갈래 (섹션 3의 뼈대)
- 👨‍👩‍👧 **가족 방문**: 3세대(조부모–부모–자녀) 만족 / 호수뷰 + 넓은 공간 + 아기의자 + 한결같은 서비스
- 💼 **비즈니스 방문**: 접대·회식 / 호수뷰 + 프라이빗 룸 + 다양한 옵션

---

## 2. 기술 & 형태

- **형태**: 원페이지 스크롤 랜딩 (섹션 5개 세로 흐름)
- **기술**: 정적 HTML + CSS + Vanilla JS (빌드 불필요, Vercel 즉시 배포)
- **언어**: 🇻🇳 베트남어 **기본** + 🇬🇧 영어 토글. JSON 사전(dictionary) 기반, 버튼 클릭 시 `data-i18n` 텍스트 일괄 교체. 선택 언어는 `localStorage` 저장
- **반응형**: 모바일 우선 (광고 유입 대부분 모바일)

---

## 3. 디자인 방향

- **톤**: 다크 프리미엄 + 골드 액센트 (광고 소재와 동일 톤 → 광고↔랜딩 일관성)
- **컬러**: 딥 차콜/블랙 배경, 골드(#C8A96A 계열) 포인트, 화이트 텍스트, 하남 레드(로고) 소량
- **타이포**: 세리프 헤드라인(고급감) + 산세리프 본문(가독성). 베트남어 성조 표기 완전 지원 폰트
- **무드**: 시네마틱 야경 호수뷰, 넉넉한 여백, 고기 클로즈업, 부드러운 스크롤 페이드인
- **품질 기준**: soft-skill / taste-skill 기준 준수 (AI 티 나는 제너릭 디자인 금지)

## 4. 섹션 구성

### Section 1 — Hero
- 풀스크린 야경 외관(5층 건물, 호수뷰) 배경 — **고화질 광고 원본 이미지** 사용
- 헤드라인: "Tinh hoa thịt nướng Hàn Quốc ngay tại Hồ Tây" / "The essence of Korean Premium BBQ at West Lake"
- 서브: 4.8★ Google · Korean No.1 BBQ
- 상단 고정 헤더: 로고 + 언어토글(🇻🇳/🇬🇧) + CTA 버튼
- 스크롤 유도 인디케이터

### Section 2 — 하남 스토리 (Our Story)
- hanampig.co.kr '하남스토리' 기반 + 회사소개 PDF 근거 → **영어 + 베트남어**
- 핵심 수치 카운터: 한국 K-BBQ 선호도 **1위(48.9%)**, 고객 만족도 **97%**, **176개** 매장, **2010년**~, 연 방문객 **467만명**
- 글로벌 확장 스토리: 말레이시아·일본·인도네시아 → **베트남(호치민·하노이 서호)**
- **"왜 베트남인가"** 블록: 인지도(Awareness) + 신규 유입(New Acquisition) 메시지 명시
- 한국 최초 돼지고기·김치 인증, 자체 R&D 부위별 2cm 정형 등 신뢰 근거

### Section 3 — 왜 하남돼지집 (광고 소재 맞춤 소개)
- 2단 카드 레이아웃:
  - **가족 카드**: 호수뷰 · 넓은 공간 · 아기의자 · 3세대 만족 · 한결같은 서비스
